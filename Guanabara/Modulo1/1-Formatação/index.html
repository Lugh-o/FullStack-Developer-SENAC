Parágrafos e quebras de linha

Você pode escrever um parágrafo de qualquer jeito. Basta colocar tudo entre <p> e </p>.

Se precisar quebrar o texto em algum lugar específico
como esse, você pode usar a tag <br>.

Vamos adicionar alguns símbolos especiais: ® © ™ € £ ¥ ¢ Δ ↑

Vamos adicionar alguns emoji: 🖖 🤓

Principais formatações

Busque utilizar sempre tags semânticas, já que o HTML5 é uma linguagem de marcação semântica

Nesta frase temos um termo em negrito usando a tag <b></b> (não semântica).

Nesta frase temos um termo em destaque usando a tag <strong></strong> (semântica)

Nesta frase temos um termo em itálico usando a tag <i></i> (não semântica).

Nesta frase temos um termo em ênfase usando a tag <em></em> (semântica).

Podemos criar também um texto marcado utilizando a tag <mark></mark>.

Estamos criando um texto grande e um texto pequeno nesse parágrafo com as tags <big></big> e <small></small> (big está obsoleta).

Podemos marcar um texto excluído para indicar que ele deve ser lido, mas não considerado com a tag <del></del>.

Existe também o sublinhado com a tag <u></u>(não semântica).

Texto sobrescrito: x20+3 com a tag <sup></sup>.

Texto subscrito: H2O com a tag <sub></sub>.
